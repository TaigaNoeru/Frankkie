## [4.1.0] - 2025-12-30

### Released as Frankkie
- Rebranded from APEX v4.1 to Frankkie
- Split into two versions: Full (48 modules) and Lite (core only)
- Created comprehensive GitHub Pages documentation

### Core Framework
- Prime Directives: Player autonomy, information boundaries, character perspective
- {{char}} perspective system with ensemble scene support
- Proactive World Mandate (Nigiyaka Principle for bustling atmosphere)
- Response format with markdown headers and typography standards

### Quality Modules (Always Active)
- Anti-Slop Core Constraints with comprehensive ban list
- Pre-Writing Planning Process (8-step internal checklist)
- Logic & Consistency Check (7 verification categories)
- Psychological Depth Framework (multi-layered motivation)
- Dialogue Standards (realistic speech patterns)
- Sensory Grounding (five-senses immersion)
- Personality Anchoring System
- Tucao Meta-Commentary (quality awareness)
- Advanced Psychology (core vs surface traits)
- Sensory Vocabulary Libraries
- Nemo Categorical Bans Expansion

### Optional Content Toggles
- Adult Content Realism framework
- Combat & Violence System with grounded mechanics
- Plot Armor (user protection mode)
- Realistic Failure (high-stakes mode)

### POV & Voice Options
- First-Person POV toggle
- Colored Dialogue System (HTML color-coding)

### Response Length Control
- SHORT (150-250 words)
- MEDIUM (400-600 words)
- LONG (700-1000 words)
- Default: 250-400 words

### Style Presets
- Minimalist (Hemingway)
- Noir/Hardboiled
- Epic Fantasy
- Horror/Dread
- Literary Fiction

### Chain-of-Thought Specialists
- Combat Specialist (spatial awareness, tactics)
- Romance/Intimacy Specialist (power dynamics, vulnerability)
- Mystery Specialist (information control, red herrings)
- Horror Specialist (dread building, isolation)

### Utility Modules
- Character Initiative Mode (proactive NPCs)
- Position & State Tracker (visual status panel)
- Simple HTML Artifacts (in-world documents)
- CYOA Mode (choice menus)
- Recap Generator (on-demand summaries)
- Genre-Specific Guidance
- Automated Consistency Tracker

### Parameters
- Temperature: 0.95 (creative variance)
- Top-K: 120 (wider vocabulary)
- Top-P: 0.95 (nucleus sampling)
- Frequency Penalty: 0.17
- Presence Penalty: 0.15
- Repetition Penalty: 1.13
- Max Tokens: 6000

## Version Naming Convention

**Format**: MAJOR.MINOR.PATCH

- **MAJOR**: Breaking changes, complete overhauls
- **MINOR**: New features, new modules, significant enhancements
- **PATCH**: Bug fixes, documentation updates, minor tweaks

**Examples**:
- `5.0.0` - Complete rewrite of core framework
- `4.2.0` - Added 3 new CoT specialists
- `4.1.1` - Fixed conflict detection bug
